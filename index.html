funciones

Una función de JavaScript es un bloque de código diseñado para realizar una tarea particular. Una función de JavaScript se ejecuta cuando "algo" la invoca (la llama).

Los nombres de funciones pueden contener letras, dígitos, guiones bajos y signos de dólar (las mismas reglas que las variables). Los paréntesis pueden incluir nombres de parámetros separados por comas: ( parámetro1, parámetro2, ... ) El código a ejecutar por la función se coloca entre llaves: {}
